# ttn-iot-explorer-board
The Things Network IoT expansion board for: 
* micro:bit
* Calliope  Mini

## Features
* LoRa Module
  * RN2483
  * EMB-LR1272 / EMB-LR1272E
* Connectivity
  * micro:bit edge connector
  * CALLIOPE MINI 26-pin connector
* Battery holder for CR2032 battery
